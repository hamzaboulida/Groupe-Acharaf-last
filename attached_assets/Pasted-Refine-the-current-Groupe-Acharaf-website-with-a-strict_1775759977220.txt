* **Primary:** #082634

## Color rules (strict)

Use this hierarchy:

* **#082634** = main background color for the global experience
* **#8EA4AF** = support color only, for subtle UI details, cards, borders, hover states, or secondary elements
* **#DCE0E7** = light contrast color only, for selected text blocks or soft contrast sections
* **#080629** and **#212F4F** = depth colors only, for gradients, overlays, and dark transitions

* Do **not** use the secondary color as a dominant section background
* Do **not** use random light blue/grey backgrounds in major sections
* Do **not** use an out-of-context footer background
