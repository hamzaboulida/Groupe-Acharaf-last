## 2. FLOATING WHATSAPP BUTTON

Add a WhatsApp floating icon/button that is always visible on the website, like a premium chat/contact assistant.

### Requirements

* fixed position on all pages
* visible on desktop and mobile
* elegant and minimal
* do not make it look cheap or aggressive
* use smooth hover animation
* use the brand design language
* open WhatsApp contact link directly

### Style

* premium floating button
* subtle shadow
* rounded and elegant
* integrated with the luxury UI, not like a default plugin

It should feel like:
👉 a premium contact shortcut
Not:
❌ a cheap chatbot widget
